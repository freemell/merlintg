## üìä **Technical Stack**

### **OG Merlin (Web)**
- **Frontend**: Next.js 15, React 19
- **AI**: Groq SDK

- **Swaps**: Jupiter Lite API
- **Database**: Prisma + SQLite
- **Security**: AES-256 encryption
